前端应用发布——devops

Mar 06 2021

简介

DevOps = development + operations。即开发和运维的组合。
DevOps 是针对企业中的研发人员、运维人员和测试人员的工作理念，是他们在应用开发、代码部署和质量测试等整条生命周期中协作和沟通的最佳实践。
DevOps 强调整个组织的合作以及交付和基础设施变更的自动化、从而实现持续集成、持续部署和持续交付，用以提高软件开发质量，并缩短软件开发生命周期。

架构

由于DevOps是一种跨职能(开发、测试、运维)的工作模式，不是一个单独的DevOps工具，因此DevOps作为工具时，它包含了多个工具，形成了一个完整的工具链。

  • 项目管理（PM）：Jira
  • 代码管理：GitLab
  • 持续集成与部署（CI&CD）：GitLab CI && jenkins（持续集成工具，用于编译、打包程序，运行单元测试）、npm、maven

    more: https://souche.yuque.com/bggh1p/8961260/tcmwan

  • 事件处理：蓝风 (egg)
  • 容器：Docker （跨平台运行，实现资源隔离，提供易用的镜像管理）
  • 编排：Kubernetes （管理 Docker 容器）
  • 容器平台： Rancher （企业级容器管理平台）
  • 镜像扫描：Clairctl
  • 服务注册与发现：etcd
  • 脚本语言：python
  • 日志管理：EFK
  • 系统监控：prometheus
  • Web服务器：Nginx
  • 数据库：MySQL redis

    参考：https://souche.yuque.com/dhkpgh/xuknga/ouog8b
