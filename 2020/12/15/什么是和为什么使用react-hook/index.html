什么是和为什么使用react hook

December 15, 2020 1548

什么是react hook

  • React 16.8带来的全新特性，即将替代class组件的写法。（react有class组件和函数式组件两种）。hook是一个特殊的函数，可以让你勾入react特性。

    使用hook解决的问题

  1. 组件很难复用状态逻辑（hook之前使用hoc或者是render Props解决）
  2. 复杂组件难以理解，尤其是生命周期函数

    什么时候会用useState Hook？

    函数式组件中想使用state时..用于记录/改变组件状态值。

    Effect Hook

  • 无需清除的副作用

    ajax请求、dom操作等

  • 需要清除的副作用

    dom事件监听等。 在effect中返回清除effect的函数

  • 控制effect的执行（解决一些性能问题）

    effect传递第二个参数（数组类型deps[]）, [] 只执行一次effect，
